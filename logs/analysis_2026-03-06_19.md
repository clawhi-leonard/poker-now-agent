## 🎯 Critical Fix Verified: Raise Slider

**BEFORE THIS SESSION:** 39% of raises showed "Raised to ? (range 2-100)" — broken slider.
**THIS SESSION:** 60/60 raises = **100% success rate!** Zero failures.

**Root Cause Found:** `click_action_button("Bet")` was matching "BET 20" (preset min-bet, acts immediately) before "BET" (opens custom slider panel). The preset "BET 20" button places a bet without showing any slider, so when the code tried to find the slider afterwards, nothing was there.

**Fix:** New `_click_raise_opener()` function that:
1. Matches buttons with EXACT text "Bet"/"Raise" (no numbers)
2. Falls back to keyboard shortcut "r" which always opens the slider panel
3. Waits up to 2.5s for the panel to render before interacting
4. Uses getBoundingClientRect() instead of offsetParent for visibility checks

## Key Issues Found

### 1. Auto-Rebuy Still Not Working
AceBot busted in 4 hands, Clawhi in 14. Neither rebuyed. Game went from 4-way to heads-up.
- **Root Cause:** The rebuy button might use different selectors than expected, or the bust detection isn't triggering.
- **Fix Priority:** HIGH — games are boring when bots die permanently

### 2. Fold Rate Too Low
- NIT folds 26% preflop → should be 40-50%
- STATION folds 0% preflop → should be at least 10-15%
- Overall fold rate 12% → should be 25-30%
- **Root Cause:** The tightness thresholds in `bot_decide()` allow too many marginal hands through. The call thresholds need to be higher.

### 3. Raise Sizes Often Too Small
Many raises are to the minimum (20 chips when range is 20-2000). The raise_amount calculation produces values like 6-15 which get clamped up to min_raise.
- **Root Cause:** `pot * random.uniform(0.4, 0.6)` on small pots produces tiny numbers
- **Fix:** Use at least 2x-3x BB for raises, not just pot-fraction

### 4. Postflop Play Improvements Needed
- NitKing bluffed 6h5s on the flop (16% equity) → NIT shouldn't be bluffing
- CallStn calling river bets with 12-16% equity → too loose even for a station
- Some bots raising tiny amounts on the river with strong hands instead of value betting
