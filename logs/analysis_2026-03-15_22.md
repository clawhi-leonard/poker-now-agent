## 🎉 Major Fix: Seating Flow
**Root cause found and fixed:** After the host approves the first bot's seat request, pokernow.com's React SPA re-renders the page, leaving the DOM in a blank/stale state. Subsequent approval attempts failed because there was nothing to click.

**Fix:** Reload the host page (`page.goto(game_url)`) after each bot approval. This gives the page a clean state with all seated players visible and any new pending requests shown inline.
